二愣子前端水平，直接拷贝的后台管理系统模板，原仓库地址， 在这个仓库地址上，试着接入自己的接口
https://github.com/PanJiaChen/vue-element-admin

## 开发
```
# install dependency
npm install

# develop
npm run dev
```

## 构建
```
# build for test environment, 对应配置文件.env.production
npm run build:stage

# build for production environment, 对应配置文件.env.staging
npm run build:prod
```
